LPM - Launchabe Packet Manager for Linux - 2026 - Originally by domesco
========================================================================

Features:
-basic package management
#-docker integration

Commands:
-lpm sync           - downloading the registry file (registry.json)
-lpm trans          - installing the registry package to the cache
-lpm rem <package>  - remove the package from the cache
-lpm osm init       - initializing the osm function
